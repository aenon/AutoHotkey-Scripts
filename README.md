# AutoHotkey scripts

AHK scripts for key tweaking on Windows. 

## numpad.ahk

remapping numpad keys to enable shortcuts

 | [       | ]       | =       | BackSp  |

 | NumLk   | ^z      | ^+z     | ^w      |

 | PrevTab | Up      | NextTab | ^t      |

 | <-      | Down    | ->      | Tab     |

 | ^x      | ^c      | ^v      | Alt     |

 | LCtrl   | LCtrl   | LWin    | Alt     |
